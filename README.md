# 2 Months To Us, Bebuuuu ❤️

This is a Vite + React starter for the anniversary experience. It includes TailwindCSS and Framer Motion and a scene-based flow.

Quick start:

```bash
npm install
npm run dev
```

Add your assets:
- `src/assets/photos/*.svg` or jpg/png
- `src/assets/music/placeholder.mp3`
- `src/assets/videos/placeholder.mp4`
